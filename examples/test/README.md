# Test/Advanced Example

This example is used for automated testing and demonstrates some of the more advanced features of the library, such as:
- Setting up a local node with a default object dictionary.
- Configuring SYNC and PDOs.
- Adding a file extension to the object dictionary.
- Using pprof for profiling.

**Note:** This example is more complex than the others and is primarily intended for development and testing purposes.

## How to Run

Make sure you have a running CAN interface (e.g., `can0` or `vcan0`).

```bash
go run main.go
```
